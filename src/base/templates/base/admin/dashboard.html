Website Info

This option is available for Admin roles.

Create: Using the navigation bar, click on the WebsiteInfo.
You will be prompted a form with details for the website, fill them and submit. All the details on the website will be set.

Update: Using the navigation bar, click on the WebsiteInfo.
You will be prompted a form with details for the website, make your changes and submit. All the details on the website will be changed.

Users

This option is available for Admin roles.

Create: Using the navigation bar, click on the Users. On top left of the page, click on Add Icon. Next, you will be prompted a user form to be filled by the user.
After filling and submitting, the user will receive a verification mail on the provided email. They should click the link to verify.

Update/View: Using the navigation bar, click on Users and search for the user you want to update. Click on their username or on the button at the far end labelled Update. It will redirect you to a limited form to make changes to a users settings.

Only users with verified emails and is_staff turned true are allowed to login the system.

Delete: Using the navigation bar, click on Users and search for the user you want to delete. Click on their username or on the button at the far end labelled Delete. It will redirect you to a limited form to make changes to a users settings or prompt you a message to confirm.

Profile

Update/View: Using the navigation bar, click on your profile at the bottom of the navigation, and click profile. Make changes to your profile and submit.

Change Password: Using the navigation bar, click on your profile at the bottom of the navigation, and click profile. At the bottom, look for a button labelled Change Password and click on it. Change the password and submit.
